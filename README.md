#### 三 项目介绍
1. 管理员功能
全流程平台管理功能：
人员与内容管理：用户管理
文创业务管理：文创产品、文创活动、文化资讯发布、定制设计
系统基础功能：系统首页
2. 用户功能
文创消费与互动功能：
基础浏览：文创产品、文创活动、文化资讯、系统首页
社交与个人管理：社交分享、个人中心、账户管理、我的收藏
业务操作：文创订单信息、文创预约信息、文创收藏信息、定制设计
该平台聚焦文创内容的推荐与服务场景，通过管理员统筹文创资源与用户管理，用户实现文创内容浏览、互动、消费与定制，实现文创产品从信息展示到个性化服务的全流程数字化，适配文创行业的内容推广与消费需求。

#### 四、项目技术
后端框架：SpringBoot++Mybatis+Mybatis-Plus

前端框架：Vue + Vue Router + ELementUI + Axios

#### 五、 运行环境
开发语言：Java

开发工具：IDEA，Eclipse,Myeclipse，VSCode都可以。推荐IDEA,VSCode

JDK版本：1.8

数据库版本: MySQL8.0.x以上

项目管理：Maven

Node版本：14


#### 留、安装教程
使用Navicat或者其它工具，在mysql中创建对应名称的数据库，并导入项目的sql文件；

使用IDEA/Eclipse导入项目，若为maven项目请选择maven;导入成功后请执行maven clean;maven install命令，然后运行；

进入src/main/resources修改application.yml里面的数据库配置

启动项目
